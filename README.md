# Loja Virtual de Materiais de Construcao

Interface de e-commerce voltada para venda de materiais de construcao, com catalogo organizado por categorias e navegacao intuitiva.

## Preview

[Ver Demo](https://loja-virtual-de-materiais-de-constr.vercel.app/)

## Tecnologias

- React
- TypeScript
- CSS
- JavaScript
- Vite

## Como rodar

```bash
npm install
npm run dev
```

## Funcionalidades

- Catalogo de produtos por categoria (madeira, eletrica, hidraulica, etc.)
- Sistema de busca e filtros
- Carrinho de compras
- Layout responsivo
- Design profissional para o segmento de construcao
